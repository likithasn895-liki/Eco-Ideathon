# Output Results

This folder contains the results generated after running
the solar panel detection inference pipeline.

## Files

- results.json  
  Contains verification results such as:
  - sample_id
  - latitude and longitude
  - solar presence (true/false)
  - buffer radius used
  - estimated solar panel area

These outputs act as digital proof for rooftop solar
verification under PM Surya Ghar Yojana.
